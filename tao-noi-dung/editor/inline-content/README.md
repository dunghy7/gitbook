# Inline Content

Có thể insert trực tiếp các Ảnh, Icon, Link hoặc phép toán trực tiếp bằng các gõ kí tự / và nó sẽ hiển thị như ảnh

Sẽ có 4 lựa chọn để insert như sau:

* Image
* Emoji
* Link
* Math & TeX
